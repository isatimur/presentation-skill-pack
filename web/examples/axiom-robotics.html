// ARCHITECTURE OVERVIEW · AX-004 · EXPLODED VIEW

Four-layer intelligence stack

Proprietary software layers — hardware-agnostic, built for warehouse-scale concurrency.

DATA FLOW: WMS ORDER → FLEET BRAIN → MOTION PLANNER → ROBOT UNIT → VERIFY & CONFIRM
01 // AX-SYS-BRAIN
AxOS Fleet Brain
Central orchestration kernel. Receives WMS order stream, dispatches pick tasks to optimal unit. 300-bot concurrency at <5ms task latency.
02 // AX-SYS-NAV
Spatial Nav Engine
Real-time 3D map fusion from LIDAR + stereo cameras. Dynamic re-routing around humans and obstacles. SLAM refresh: 40ms.
03 // AX-SYS-VISION
Pick Vision System
SKU recognition via custom CV model (AxVision-v3). Handles 240K+ SKU catalog with <80ms inference time per pick event on-device.
04 // AX-SYS-VERIFY
Verify & Audit
Weight + barcode dual-confirm before release. Error rate: 3 per 10,000 picks. Full audit trail to WMS within 200ms of pick completion.
