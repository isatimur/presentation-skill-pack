What it does

presentation-skill-pack gives Gemini CLI a schema-validated workflow for authoring slide decks. You describe what you want in plain language — or paste rough notes — and Gemini CLI fills a structured Deck JSON spec. A renderer converts that spec into a single, self-contained HTML file: open in any browser, no hosting, no build step.

Full mode also wires the MCP server for typed tool calls within Gemini CLI agent workflows.

Output is portable. You own the file — email it, drop it on a CDN, or open it locally. It doesn't live on any platform.

Install

1

Run the installer

npx @presentation-skill-pack/install gemini-cli

Or lite mode (skill only, no MCP server):

npx @presentation-skill-pack/install gemini-cli lite

Files land in: ~/.gemini/instructions/ (GEMINI.md skill file)

2

Restart Gemini CLI

Restart Gemini CLI and ask: "Build a product demo deck from these notes…"

3

Ask for a deck

Build a 10-slide pitch deck from these notes:
[paste your notes here]

The agent writes the Deck JSON spec, renders it, and saves the HTML file to your working directory.
