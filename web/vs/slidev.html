presentation-skill-pack
vs Slidev

Slidev requires you to author. presentation-skill-pack lets your agent author.

Side-by-side

Criterionpresentation-skill-packSlidev
Where it runsInside your AI agent (Claude Code, Cursor, Copilot, Codex, Gemini CLI)Markdown-based dev presentation framework

Where Slidev is stronger

  • Code-friendly (syntax highlighting, live demos)
  • Git-versionable
  • Strong dev community
  • Highly customizable Vue components

Where presentation-skill-pack is stronger

  • Agent writes the spec — you just describe what you want
  • No dev server needed — single HTML file opens anywhere
  • Works inside Claude Code, Cursor, Copilot, Codex, Gemini CLI
  • Free, MIT licensed

Verdict

Slidev is excellent for technical talks where you hand-author every slide. presentation-skill-pack is for when you want an AI agent to generate the deck from notes or a brief — less control, much less effort.

Try presentation-skill-pack in one command:

npx @presentation-skill-pack/install claude-code
