# Pricing — presentation-skill-pack

## Free (the only tier)

- **Price:** $0/month — forever free
- **License:** MIT open source
- **No account required**
- **No usage limits**

### What's included

- Full skill for all supported agents (Claude Code, Cursor, Codex, Gemini CLI, GitHub Copilot)
- MCP server with 5 typed tools: `render_deck`, `list_themes`, `get_theme`, `validate_deck`, `create_deck`
- Node.js and Python renderers
- 5 built-in themes (default-tech, corporate, playful, luxury-minimalist, retro-arcade)
- Self-contained HTML output (no hosting required)
- npm + PyPI packages

### Install

```
npx @presentation-skill-pack/install <agent>
```

Replace `<agent>` with: `claude-code`, `cursor`, `copilot`, `codex`, or `gemini-cli`
